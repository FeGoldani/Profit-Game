Instruções

Este é um jogo no qual o objetivo é ensinar de maneira educativa o jogador a gerenciar o seu dinheiro. Percorrendo o caminho dos tijolos amarelos, o jogador deve investir nos objetos que aparecem no caminho para agregar valor ($). Cada disparo vai influenciar nos resultado dos futuros investimentos. O objetivo do jogo é somar a maior quantidade de $ durante as fases.

Comandos

Utilize as setas do teclado para mover a nave. A tecla de espaço efetuará os investimentos nos objetos que aparecem pelo caminho.
